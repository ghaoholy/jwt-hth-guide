# jwt-hth-guide

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内包含多个自包含的网页文件，每个页面均为独立实现，不依赖于特定的域名或外部网站。本仓库主要用于整理、保存以及方便查阅各类 HTML 页面资源。

## 目录结构

```
jwt-hth-guide/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
└── README.md       # 本说明文件
```

- `pages/`：每个 HTML 文件为一个独立的页面，文件名通常反映其内容或用途。
- `assets/`：存放页面所需的公共或专用资源文件，按需引用。

## 页面归档说明

- 所有页面均为纯前端实现，无需后端服务支持，可直接在浏览器中打开运行。
- 页面内容可能包含演示、示例、工具或说明性资料，不指向任何具体网站或服务。
- 归档的页面文件可能会不定期更新或补充，以保持内容的完整性和可用性。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/jwt-hth-guide.git
   ```
2. 打开 `pages/` 目录下的任意 HTML 文件，即可在浏览器中查看对应页面内容。

## 维护说明

- 本仓库由维护者根据实际需要更新页面内容。
- 欢迎通过 Issue 或 Pull Request 提交改进建议或补充新页面。
- 提交的页面应保持独立、无外部依赖（或依赖已包含在 `assets/` 中），并符合归档规范。
